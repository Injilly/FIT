# 💪 Fitness Interactive Tracker

Fitness Interactive Tracker is a simple fitness website created as a school project.
It allows users to explore workout plans, calculate BMI, track workout schedules, and interact with a basic fitness chatbot.

The website is designed to look like a mobile fitness app with a neon pink aesthetic theme, glass-style cards, and animated background effects.

---

📌 Features

🏠 Home Page

- Motivation generator
- Fitness progress chart
- Aesthetic UI design

🏋️ Workout Planner

- Workout calendar scheduler
- BMI calculator
- Random workout suggestion generator
- List of 100 workouts with sets and reps

🤖 CoachBot

- Simple chatbot for fitness motivation
- Basic responses to encourage workouts

ℹ️ About Page

- Information about the project
- Contact information (sample only)
